### Quick start <a name="start"></a>
#### 1. Straight install.
##### 1.1 Install torch with cuda.
```bash
pip3 install -U torch --extra-index-url https://download.pytorch.org/whl/cu113
```
##### 1.2 Install opennn.
```bash
pip install -U opennn
```
#### 2. Dockerfile.
```bash
cd docker/
docker build .
```
